# Weboldal Frontend

## Letöltés és futtatás
1. A vizsgaremeket töltse le a public repositryból
2. Nyisson egy terminált a Vizsgaremek mappában és cd {mappa neve} paranccsal menjen vegég a mappákban amíg a mozifront mappában nem találja magát vagy nyisson egy terminált egyenesen a mozifront mappából
3. A mozifront terminálba írja be az alábbi parancsot: "npm i axios react jwt-decode react-dom react-router-dom" majd írja be "npm run dev"
4. Ha ezeket megfelelően elvégezte akkor a terminálban ennek kéne megjelennie "Local:   http://localhost:5173/".
5. Nyissa meg és onnan szabad a pálya
